# 한컴오피스 한글 수식 편집기 명령어 완전 목록

한글(HWP) 수식 편집기는 **약 260여 개의 고유 명령어**를 지원하며, LaTeX와 유사하되 백슬래시(`\`) 없이 키워드를 직접 입력하는 독자적 문법 체계를 사용한다. 이 목록은 한컴 공식 도움말(help.hancom.com)과 한글문서파일형식 수식 명세서(revision 1.3, 2018)를 기반으로, 모든 명령어를 카테고리별로 정리한 것이다. 수식 편집기는 `Ctrl+N, M`으로 실행하며, `Shift+ESC`로 종료한다.

---

## 입력 규칙과 서식 제어 문자

수식 편집기의 스크립트 창에서 일반 공백·Enter·Tab은 단순 구분자 역할만 하며, 출력에는 반영되지 않는다. 실제 수식 출력에 공백이나 줄바꿈을 넣으려면 아래 서식 문자를 사용해야 한다.

| 입력 문자 | 기능 |
|-----------|------|
| `~` | 1칸 공백 (full space) |
| `` ` `` (백틱) | ¼칸 공백 (quarter space) |
| `{ }` | 여러 항목을 하나의 단위로 묶기 |
| `" "` | 9자 이상 단어를 하나의 단위로 묶기 |
| `#` | 줄바꿈 (Enter 효과) |
| `&` | 열 정렬 (Tab 효과) |

**대소문자 규칙:** 대부분의 명령어는 대소문자를 구분하지 않으나, 그리스 문자(대문자 `Alpha` vs 소문자 `alpha`), 화살표(소문자 `larrow` → 단일선, 대문자 `LARROW` → 이중선), 일부 함수(`lim` vs `Lim`, `exp` vs `Exp`)는 반드시 표기를 맞춰야 한다.

## 기본 구조 명령어: 분수, 첨자, 루트

이들은 수식 편집기에서 가장 빈번하게 사용되는 핵심 명령어다. `OVER`로 분수를, `^`와 `_`로 첨자를, `SQRT`로 근호를 만든다.

| 명령어 | 기능 | 사용 예 |
|--------|------|---------|
| **`OVER`** | 분수 (가로선 있음) | `{a+b} over {c+d}` |
| `ATOP` | 분수 형태 (가로선 없음) | `x atop y` |
| **`^`** 또는 `SUP` | 위첨자 | `E=mc^2` |
| **`_`** 또는 `SUB` | 아래첨자 | `H_2 O` |
| `LSUB` | 왼쪽 아래첨자 | `x LSUB y` |
| `LSUP` | 왼쪽 위첨자 | `x LSUP y` |
| **`SQRT`** | 제곱근 | `sqrt {x^2+1}` |
| `ROOT n OF A` | n제곱근 | `root {3} of {x}` |
| `^n SQRT` | n제곱근 (대체 문법) | `^3 sqrt {x^2+1}` |
| `TIMES` | 곱셈 기호 (×) | `2 times 5` |
| `NOT` | 다음 기호에 사선 긋기 (부정) | `not =` |
| `BIGG` | 다음 기호 확대 | `bigg /` |

## 적분, 급수, 극한 명령어

대형 연산자(large operator)에는 `_`(하한)와 `^`(상한)로 범위를 지정한다. 예: `int _0 ^1 {f(x)}dx`.

### 적분 기호

| 명령어 | 기호 | 설명 |
|--------|------|------|
| **`INT`** | ∫ | 단일 적분 |
| `OINT` | ∮ | 선적분 (폐곡선) |
| `DINT` | ∬ | 이중 적분 |
| `TINT` | ∭ | 삼중 적분 |
| `ODINT` | ∯ | 폐곡면 이중 적분 |
| `OTINT` | ∰ | 폐곡면 삼중 적분 |

### 합·곱·집합 연산자 (범위 표시 가능)

| 명령어 | 기호 | 설명 |
|--------|------|------|
| **`SUM`** | Σ | 합 (시그마) |
| **`PROD`** | ∏ | 곱 (파이) |
| `COPROD` | ∐ | 여곱 (coproduct) |
| `UNION` | ⋃ | 합집합 (대형, 범위 표시) |
| `INTER` | ⋂ | 교집합 (대형, 범위 표시) |

`SMALL` 접두어를 붙이면 범위 위치 없이 기호만 출력된다: **`SMALLUNION`**(∪), **`SMALLINTER`**(∩), **`SMALLPROD`**, **`SMALLSUM`**.

### 극한

| 명령어 | 설명 | 사용 예 |
|--------|------|---------|
| `lim` | 극한 (소문자 l) | `lim _{x -> 0} {1 over x}` |
| `Lim` | 극한 (대문자 L) | `Lim _{n -> inf}` |

## 괄호와 구분 기호

`LEFT`/`RIGHT`를 붙이면 괄호가 내용 높이에 맞춰 **자동으로 크기 조절**된다. 분수나 행렬처럼 높은 수식을 감쌀 때 반드시 사용해야 한다.

| 명령어 | 출력 |
|--------|------|
| `LEFT( ... RIGHT)` | 자동 크기 소괄호 ( ) |
| `LEFT[ ... RIGHT]` | 자동 크기 대괄호 [ ] |
| `LEFT{ ... RIGHT}` | 자동 크기 중괄호 { } |
| `LEFT| ... RIGHT|` | 자동 크기 절댓값 기호 \| \| |
| `vert` | 단일 세로선 \| |
| `VERT` | 이중 세로선 ‖ |

## 행렬 명령어

행렬 내부에서 열은 `&`로, 행은 `#`으로 구분한다. 예: `pmatrix{a&b # c&d}`.

| 명령어 | 설명 |
|--------|------|
| **`MATRIX`** | 괄호 없는 행렬 |
| **`PMATRIX`** | 소괄호 행렬 ( ) |
| **`BMATRIX`** | 대괄호 행렬 [ ] |
| **`DMATRIX`** | 세로선 행렬 \|\| (행렬식) |

열 기반 입력 시 `col`(가운데 정렬), `lcol`(왼쪽 정렬), `rcol`(오른쪽 정렬)을 사용할 수 있다: `matrix{ col{a#b} col{c#d} }`.

## 꾸밈(장식) 명령어

문자 앞에 명령어를 쓰면 해당 문자 위(또는 아래)에 장식이 붙는다. `hat`, `check`, `tilde`, `bar`, `arch`는 여러 문자에 걸쳐 **자동으로 늘어난다** (예: `hat AA`, `hat AAA`).

| 명령어 | 효과 | 사용 예 |

| `acute` | 양음 부호 (´) | `acute A` → Á |
| `grave` | 억음 부호 (`) | `grave A` → À |

## 폐기 예정(레거시) 명령어

한/글 97에서 사용되었으나 현행 버전에서는 `_`와 `^`로 대체된 명령어다. 파일 호환성을 위해 내부적으로 여전히 인식될 수 있다.

| 명령어 | 과거 기능 | 현행 대체 |
|--------|----------|----------|
| `FROM` | 하한 범위 | `_` (아래첨자) |
| `TO` | 상한 범위 | `^` (위첨자) |
| `PHANTOM` | 보이지 않는 자리 차지 | (삭제됨) |

## 실전 활용 예시로 보는 문법 요약

```
# 이차방정식의 근의 공식
x = {-b plusminus sqrt {b^2 - 4ac}} over {2a}

# 정적분
int _0 ^1 {x^2}dx = {1 over 3}

# 급수
sum _{n=1} ^{inf} {1 over {n^2}} = {pi^2 over 6}

# 행렬
A = pmatrix{1&0&0 # 0&1&0 # 0&0&1}

# 조건식 (절대값 함수)
|x| = cases {x & rm if~ x >= 0 # -x & rm if~ x < 0}

# 벡터·편미분
vec F = PARTIAL over {PARTIAL x} hat i + PARTIAL over {PARTIAL y} hat j

# 극한
lim _{x -> 0} {sin x over x} = 1
```

## 결론: 체계를 알면 수식 입력이 빨라진다

한글 수식 편집기의 명령 체계는 크게 **구조 명령어**(OVER, SQRT, MATRIX, CASES 등), **대형 연산자**(INT, SUM, PROD 등), **기호 명령어**(그리스 문자, 화살표, 관계·논리 기호), **꾸밈 명령어**(hat, bar, vec 등), **서식 명령어**(rm, bold, SCALE, COLOR)의 다섯 축으로 구성된다. LaTeX 사용자라면 문법이 매우 유사하되 백슬래시가 없다는 점만 기억하면 즉시 적응할 수 있다. `LEFT`/`RIGHT`로 괄호 자동 크기 조절, `{}`로 그룹화, `#`과 `&`로 줄바꿈·열 정렬이라는 세 가지 핵심 규칙만 숙지하면, 위 260여 개 명령어를 조합하여 거의 모든 수학·과학 수식을 표현할 수 있다.
